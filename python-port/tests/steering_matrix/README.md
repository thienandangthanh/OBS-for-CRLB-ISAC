# Steering Matrix Tests

This directory contains tests for the steering matrix utilities in `utils/steering_matrix.py`.

## Test Files

- `test_steering_matrix.py` - Main test comparing Python implementation with MATLAB reference
- `test_validation.py` - Tests for input validation functions
- `test_steer_matrix_data.mat` - MATLAB reference data (generated by MATLAB script)
- `steer_matrix_comparison_case*.png` - Visualization outputs
- `steer_matrix_test_results.mat` - Test results

## Running Tests

### Prerequisites

1. Generate MATLAB reference data by running `test_steer_matrix_generate_data.m` in MATLAB
2. Ensure the MATLAB data file exists at `../../../OBS_for_CRLB_ISAC/tests/test_steer_matrix_data.mat`

### Running the Tests

```bash
# From the steering_matrix directory
python test_steering_matrix.py
python test_validation.py

# Or from the project root
python -m pytest python-port/tests/steering_matrix/
```

## Test Structure

The main test (`test_steering_matrix.py`) compares the Python implementation against MATLAB reference data for three test cases:

1. **Case 1**: Small 3x3 array with 2 angles
2. **Case 2**: Medium 4x4 array with random angles  
3. **Case 3**: Larger 5x4 array with 3 angles

Each test case verifies:
- Steering matrix `A`
- Elevation derivative `dAtheta`
- Azimuth derivative `dAphi`

The validation test (`test_validation.py`) checks input validation for:
- Angle shape consistency
- Array dimension validity
- Angle range constraints
